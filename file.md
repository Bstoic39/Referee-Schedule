# Referee Schedule Tracker PWA

A Progressive Web App for tracking referee games and earnings.

## Features
- Add and manage referee games
- Track earnings by game
- Calendar view
- Automatic archiving of past games
- Push notifications (when supported)
- Offline functionality
- Mobile-optimized

## Deployment to GitHub Pages

1. Create a new repository on GitHub
2. Upload all files to the repository
3. Go to Settings > Pages
4. Select "Deploy from a branch" 
5. Choose "main" branch and "/ (root)"
6. Your app will be available at `https://yourusername.github.io/repository-name`

## Local Testing

1. Serve the files using a local web server (required for PWA features)
2. Python: `python -m http.server 8000`
3. Node.js: `npx serve .`
4. Visit `localhost:8000` in your browser

## PWA Installation

Once deployed, users can install the app by:
1. Visit the URL in Chrome/Safari on mobile
2. Tap "Add to Home Screen" when prompted
3. Or use the browser menu > "Install App"
